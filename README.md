## 🎯 How to Run Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/jontypatel1107/Games-By-Jonty.git
   ```

2. **Navigate into the project folder**
   ```bash
   cd Games-By-Jonty
   ```

3. **Open any game** by simply opening its `index.html` file in your browser — no server or dependencies needed!

## 📖 Game Descriptions

### 🔨 Whack-a-Mole
A classic arcade-style reflex game. Moles pop up randomly from holes and your goal is to click/whack them as fast as possible before they disappear. Try to beat your high score!

### ⌨️ Typing Speed Test
A typing challenge that measures your **WPM (Words Per Minute)** and accuracy. A random passage is displayed and you race against the clock to type it correctly. Great for improving your typing skills!

### ❌ Tic Tac Toe
The timeless **X vs O** strategy game for two players. Take turns marking the grid and be the first to get three in a row — horizontally, vertically, or diagonally — to win!

### 🃏 Memory Flip Card
A memory-based card matching game. Cards are placed face-down and you flip two at a time, trying to find matching pairs. Challenge yourself to clear the board in the fewest moves!
